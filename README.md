# 🎮 Merlin Agency — Ludum SAE 2024

## 🧠 Présentation

Merlin Agency est un prototype de jeu développé dans le cadre d’un projet Ludum SAE en 2024.

Le jeu plonge le joueur dans un univers magique où il doit comprendre et utiliser différentes mécaniques pour interagir avec son environnement et progresser.

L’objectif est d’expérimenter avec les éléments du jeu, d’en comprendre les règles et de trouver les bonnes interactions pour avancer.

---

## 🎮 Gameplay

Le joueur doit :

- explorer un environnement interactif  
- comprendre les mécaniques proposées  
- tester différentes interactions  
- utiliser les bons éléments pour progresser  

Le gameplay repose sur l’expérimentation, la compréhension des règles et l’adaptation aux systèmes du jeu.
